# TEi-News 專案說明（給 Claude 看的）

## 專案概要

TEi Composites 國際新聞簡報網站。透過 PowerShell 腳本（`fetch_news.ps1`）
抓取國際新聞、用 DeepL 翻譯，產生 `index.html` 給公司內部閱讀。
排程由 GitHub Actions（`.github/workflows/`）每 10 分鐘自動執行一次。

## 互動規則

### 法務問題務必標註司法管轄區

當使用者詢問任何法務 / 法律 / 合約 / 法規相關問題時，回答前必須先明確標示
答案適用的 **國家與司法管轄區**（例如：美國聯邦法、加州法、歐盟 GDPR、
中華民國法律、中國大陸法等）。

預設使用者位於 **台灣**，適用 **中華民國法律**。若答案是以其他國家法律為
基礎（例如英美法、歐盟法），必須明確說明，並指出與台灣法律可能的差異或
告知「台灣法另有規定，建議洽台灣執業律師確認」。

不要在未標註管轄區的情況下直接給法律答案。
