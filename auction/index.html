歡迎來到 櫻拍 SakuraBid

輸入想找的物品（中文或日文），App 會自動翻譯並同時搜尋 Yahoo 拍賣、JDirectItems 與 Mercari，依結束時間／價格排序。喜歡的可加入願望清單，再一鍵帶到比比昂代購。

♡ 願望清單

⚙ 設定

翻譯

使用 MyMemory 免費翻譯 API（每日 5000 字）。可自填 DeepL／Google API Key 取得更準確結果。

搜尋來源

瀏覽器無法直接讀取日本拍賣站（CORS 限制），需透過代理。預設 allorigins 已可讀取，若失敗會回退為「於原站開啟」。

比比昂代購

點卡片上「比比昂」按鈕時會帶入該商品連結至比比昂代購頁。

匯率（顯示用）

跨裝置同步

電腦、手機之間同步願望清單與分類。可匯出 JSON 後在另一裝置匯入；或填入 JSONBin / Firebase REST 端點自動同步。
